    Generador automático de Orígenes del Software para Ubuntu

    Publicado por Guillermo Latorre el 19 de Enero de 2010

    El sistema de repositorios de los sistemas GNU/Linux es una auténtica gozada, de eso no hay ninguna duda. En Ubuntu, seguramente la distribución más popular del momento, los llaman “Orígenes del software“. Se trata de servidores remotos que alojan software de diferentes tipos para que los usuarios puedan instalar o actualizar los programas de forma totalmente automática.

    En Ubuntu existen varias alternativas para conseguir instalar aplicaciones mediante repositorios, de forma sencilla y cómoda, aunque no sean software oficialmente mantenido por Canonical. Una de ellas es Ubuntu Tweak, que consiste en una ventana donde configurar muchos aspectos del escritorio que el sistema no permite por defecto, además de permitir marcar algunas aplicaciones muy populares para instalarlas automáticamente. Desde la última versión Ubuntu Tweak también permite instalar repositorios de software de terceros, para estar siempre al día de todas las actualizaciones.

    Otra alternativa es utilizar este generador de Orígenes del Software. El funcionamiento es sencillo: el usuario va marcando los repositorios que quiere tener activados, ya sean repos oficiales o de terceros, y al pulsar sobre el botón de generación la web permitirá copiar el contenido del fichero sources.list. Ese texto hay que escribirlo en el /etc/apt/sources.list, eliminando el que ya existe (mejor hacer una copia de seguridad del fichero, por si las moscas). Automáticamente ya se tendrá acceso a los repositorios seleccionados.

    Además, posiblemente tendrás que descargar también las claves de verificación de cada repositorio, pero el generador te lo pone muy fácil y te muestra una línea de comandos para copiar y ejecutar en un terminal para obtenerlas automáticamente.

    Si echas un vistazo a la web del generador, la verdad es que se han pegado una buena currada recopilando repositorios de aplicaciones populares, como puede ser AWN, Banshee, Chromium,  DropBox, los repos oficiales de Google y GetDeb, Medibuntu, Skype, VirtualBox…
