    Instalando el Gnome Main Menu de SUSE

    Publicado por Guillermo Latorre el 25 de Agosto de 2006

    Actualizado (7-03-2007): a partir de la versión Edgy de Ubuntu no hace falta añadir ningún repositorio, sólo hace falta tener activados los universe y multiverse. El paquete se llama gnome-main-menu, sólo tienes que instalarlo mediante apt-get.

    Después de instalar el Ubuntu System Panel completamos la gama de menús para Gnome con el que trae la nueva versión de SUSE Linux.

    Si ya tenemos el USP no tenemos que tocar nada de los repositorios, pero si no lo instalamos, tenemos que hacer lo que ya se comentó: editar el fichero con los repositorios, añadir uno nuevo y actualizar las claves GPG:

    $:/>sudo gedit /etc/apt/sources.list

    Añadimos la línea:

    deb http://xgl.compiz.info/ dapper main

    Guardamos los cambios y ejecutamos:

    $:/>wget http://xgl.compiz.info/quinn.key.asc -O - | sudo apt-key add - && sudo apt-get update

    El paquete gnome-main-menu es el que nos interesa, así que lo instalamos:

    $:/>sudo apt-get install gnome-main-menu gcontrol

    Cuando termine de instalar los paquetes, hacemos click derecho en el panel en el que queramos añadir el nuevo menú y vamos a Añadir al Panel… En la sección Miscelánea seleccionamos Menú principal y lo añadimos.

    Y ahí lo tenemos:

    El menú es muy chulo, aunque una cosa que no me gusta es que no te permita directamente cambiar los comentarios de cada lanzador, que aparecen directamente ahí.

    El buscador que aparece es el propio Beagle integrado en el menú, lo cual lo hace muy muy interesante. Lo de este buscador no tiene nombre, es una maravilla trabajar con él.
