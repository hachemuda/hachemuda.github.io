    Instalando el Ubuntu System Panel (USP)

    Publicado por Guillermo Latorre el 19 de Agosto de 2006

    Vía bootlog he sabido lo sencillo que es instalar el nuevo panel de Gnome que trae la nueva versión de la distribución SUSE Linux. Los paquetes necesarios están en un repositorio para Ubuntu Dapper, por lo que con añadirlo a la lista podemos instalar directamente el panel.

    Además, podemos instalar también el Ubuntu System Panel (USP), el cual muestro en este artículo.

    Tenemos que añadir un repositorio nuevo, por lo que hay que editar (con permisos de superusuario) el fichero que contiene la lista de repositorios a los que queremos tener acceso para instalar paquetes:

    $:/>sudo gedit /etc/apt/sources.list

    En ese fichero, añadimos la siguiente línea y guardamos los cambios:

    deb http://xgl.compiz.info/ dapper main

    Para agregar las claves GPG del nuevo repositorio, ejecutamos en el terminal:

    $:/>wget http://xgl.compiz.info/quinn.key.asc -O - | sudo apt-key add -

    Actualizamos ya la lista de repositorios:

    $:/>sudo sudo apt-get update

    Y ya tenemos añadido y preparado el nuevo repositorio. Sólo queda instalar los nuevos paquetes:

    $:/>sudo apt-get install gcontrol usp

    El paquete gcontrol es para el nuevo panel de control y el paquete usp es el propio Ubuntu System Panel.

    Cuando termine de instalar los paquetes, hacemos click derecho en el panel en el que queramos añadir el nuevo menú y vamos a Añadir al Panel…. En la sección Miscelánea seleccionamos Ubuntu System Panel y lo añadimos.
