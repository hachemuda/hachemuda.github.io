Te recomiendo que lo pruebes.</p> <p><img src="/uploads/2007/06/buddylist.jpg" alt="Buddy List" /> Por fin tengo un cliente de mensajería instantánea que <strong>me convence</strong> (<a href="http://www.amsn-project.net/" title="Página oficial">aMSN</a> y <strong>Gaim</strong> no me terminaban de gustar por diferentes razones). Lo único que echo de menos, y que no sé si es que no lo he encontrado, es la opción de que la ventana minimizada parpadee en el panel cuando alguien me escribe.</p> <p>Ya sé cómo hacer que parpadee la ventana en el panel cuando alguien te dice algo: *Complementos -> Notificación de mensajes -> Insertar el número de mensajes nuevos… *(Gracias, <a href="http://gadiempire.blogspot.com/">gadi</a>).</p> <p>Algo que me ha gustado mucho son los <strong>complementos</strong>. Puedes elegir los <em>add-ons</em> que quieres activos y configurar cada uno de ellos. Supongo que dentro de poco también se podran instalar nuevos.</p> <p><strong>Instalación:</strong></p> <p>Para <a href="http://www.ubuntu.com/" title="Página oficial">Ubuntu Feisty</a>, puedes descargar <a href="http://www.getdeb.net/release.php?id=955" title="GetDeb: pidgin">el paquete .deb desde el portal GetDeb</a>. Guárdalo en tu disco duro y haz <strong>doble click</strong> sobre él para instalarlo.</p> <p>Si tienes algún problema para conectar con tu cuenta de MSN o de Google Talk, ejecuta el siguiente comando y reinicia Pidgin:<br /> <code>sudo apt-get install libnss-dev libssl-dev</code></p> <p>Si prefieres instalarlo* a mano*, te recomiendo que sigas <a href="http://jhcore.com/2007/06/04/install-pidgin-in-ubuntu/" title="jhcore.com: Install Pdgin in Ubuntu">el tutorial de jhcore</a>.</p> <p>Para otras distribuciones, pásate por la <a href="http://pidgin.im/pidgin/download/" title="Pidgin: downloads">sección de descargas</a> de la página oficial de Pidgin.</p> <p>Enlaces:</p> <ul> <li><a href="http://pidgin.im/pidgin/home">Pidgin: página oficial</a></li> <li><a href="http://www.getdeb.net/release.php?id=955">GetDeb Portal: Pidgin package</a></li> <li><a href="http://jhcore.com/2007/06/04/install-pidgin-in-ubuntu/">jhcore.com: install Pidgin in Ubuntu</a></li> <li><a href="http://www.nulleando.com.ar/index.php/2007/05/13/problemas-con-pidgin-ssl-tsl-msn-gtalk-en-ubuntu/">nulleando: problemas en Pidgin</a></li> </ul>
