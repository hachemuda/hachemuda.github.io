el foro de OpenCompositing</a>, pero lleva un tiempo caído y no se puede acceder a él. <a href="http://hachemuda.com/2007/06/18/presentando-compcomm-fusion-berylcompiz-y-sus-nuevos-plugins/" title="HacheMuda: Presentando CompComm">En el post de ayer</a> pudiste ver un video de <strong>CompComm</strong> <strong>y sus efectos</strong> en acción, así que ahora toca instalarlo y probarlo tú mismo.</p> <p>Esto es un <strong>tutorial de instalación de CompComm</strong> (<em>Composite Community</em>), que es el proyecto formado por la unión de <a href="http://www.beryl-project.org/" title="Página oficial">Beryl</a> y <a href="http://compiz.org/" title="Página oficial">Compiz</a>. El proceso es muy sencillo gracias a un <strong>script de instalación</strong> que salió del mismo foro y ahora cuelgo yo aquí para que se pueda descargar sin problemas. El proceso de instalación funciona tanto para <a href="http://www.gnome.org/" title="Página oficial">Gnome</a> como para <a href="http://www.kde.org/" title="Página oficial">KDE</a>, aunque yo sólo lo he probado en <a href="http://www.gnome.org/" title="Página oficial">Gnome</a>.</p> <p><img src="/uploads/2007/06/terminal.jpg" alt="Terminal añadiendo efectos de CompComm" /></p> <p><strong>Vamos al tema:</strong></p> <ol> <li>Descarga el script <a href="http://hachemuda.com/ficheros/makeccc" title="Descarga el script desde HacheMuda.Com">makeccc</a> <em>(click derecho -> guardar enlace como…) *y guárdalo en tu disco duro con el nombre *makeccc</em>.</li> <li>No sé si es necesario o no, pero yo mejor desinstalaría Beryl antes de nada:<br /> <code>sudo apt-get remove --purge beryl beryl-plugins beryl-manager</code></li> <li>Abre un terminal y accede al directorio donde has descargado el script. Dale permisos de ejecución:<br /> <code>chmod x makeccc</code></li> <li>Instalar los paquetes necesarios:<br /> <code>./makeccc packages</code></li> <li>Descargar/actualizar Compiz/CompComm:<br /> `./makeccc clone&#8220;`</li> <li>Instalarlo:<br /> <code>./makeccc install</code></li> <li>Si quisieras desinstalarlo puedes hacerlo ejecutando:<br /> <code>./makeccc uninstall</code></li> <li>Configurar CompComm. Tienes dos opciones (yo uso la primera). Una, ejecutando el comando:<br /> <code>ccsm</code> (o desde <em>Sistema -> Preferencias -> CompizConfig Settings Manager</em>)<br /> La segunda, ejecutando el comando:<br /> <code>ccs-settings</code> (o desde <em>Sistema -> Preferencias -> CCS Settings Manager</em>)</li> </ol> <p>En el panel del <strong>CompizConfig Settings Manager</strong> puedes configurar absolutamente todos los parámetros que quieras. Están bastante bien ordenados y accesibles, seguro que te pasas un ratito cambiando y curioseando cosas, <img src="http://www.hachemuda.com/img/icon_smile.gif" alt=":)" /> </p> <p><img src="/uploads/2007/06/compiz-settings-manager.jpg" alt="Compiz Settings Manager" /></p> <p>Ya puedes ver tu nuevo <strong>escritorio con CompComm funcionando</strong>. Abre un terminal y ejecuta:</p> <p><code>compiz-manager</code></p> <p><strong>¡Y a disfrutar!</strong> Si quieres que se active CompComm siempre que inicies sesión, no tienes más que añadir el comando <code>compiz-manager</code> como <strong>programa al inicio</strong> (en Gnome desde <em>Sistema -> Preferencias -> Sesiones</em>).</p> <p><strong>Mucho cuidado:</strong> no actives el plugin de GLib (en la sección <em>Utility</em>) porque debe estar dando muchos problemas con el gestor de preferencias.</p> <p>Relacionado:</p> <ul> <li><a
