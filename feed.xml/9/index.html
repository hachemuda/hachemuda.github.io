Partiendo de las mismas premisas que CSS Zen Garden, <a href="http://twitter.com/#!/jacobrask">Jacob Rask</a> decidió hace unos meses publicar el proyecto <a href="http://css1k.com/">CSS1K</a>: un sitio web con contenido fijo y diferentes diseños con <strong>hojas de estilo que no superan el Kilobyte de peso</strong>.</p> <p>Se trata de una demostración de fuerza, un variado ejemplo de qué puede conseguirse con muy poco código CSS.</p> <p>Las <strong>normas de participación</strong> son sencillas:</p> <ul> <li>Puedes enviar solamente CSS para la plantilla HTML común.</li> <li>Puedes enviar CSS comprimido de como mucho 1K (1024 bytes) en total.</li> <li>Los prefijos de los diferentes navegadores no cuentan dentro del peso final.
