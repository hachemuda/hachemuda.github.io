apt-get install avant-window-navigator-svn</code></p> <p>Para <strong>conseguir el efecto 3D</strong> del dock tienes que ejecutar el comando <code>gconf-editor</code>, desplegar el directorio <code>apps -&gt; avant-window-navigator</code> y en el directorio <code>bar</code> cambiar el valor de <code>bar_angle</code> a 45, y el de <code>icon_offset</code> a 15.</p> <p>Para <strong>añadir
