En Ubuntu, seguramente la distribución más popular del momento, los llaman “<strong>Orígenes del software</strong>“. Se trata de servidores remotos que alojan software de diferentes tipos para que los usuarios puedan instalar o actualizar los programas de forma totalmente automática.</p> <p>En Ubuntu existen varias alternativas para conseguir <strong>instalar aplicaciones mediante repositorios</strong>, de forma sencilla y cómoda, aunque no sean software oficialmente mantenido por Canonical. Una de ellas es <a href="http://hachemuda.com/2010/01/screencast-ubuntu-tweak-0-5-ahora-con-gestor-de-aplicaciones-y-repositorios/">Ubuntu Tweak</a>, que consiste en una ventana donde configurar muchos aspectos del escritorio que el sistema no permite por defecto, además de permitir marcar algunas aplicaciones muy populares para instalarlas automáticamente. Desde la última versión Ubuntu Tweak también permite instalar repositorios de software de terceros, para estar siempre al día de todas las actualizaciones.</p> <p><img src="/uploads/generador_origenes1.jpg" alt="Generador de Orígenes del Software" title="Generador de Orígenes del Software" /></p> <p>Otra alternativa es utilizar este <a href="http://repogen.simplylinux.ch/">generador de Orígenes del Software</a>. El funcionamiento es sencillo: el usuario va <strong>marcando los repositorios que quiere tener activados</strong>, ya sean repos oficiales o de terceros, y al pulsar sobre el botón de generación la web permitirá copiar el contenido del fichero sources.list. Ese texto hay que escribirlo en el <em>/etc/apt/sources.list</em>, eliminando el que ya existe (mejor hacer una copia de seguridad del fichero, por si las moscas). Automáticamente ya se tendrá acceso a los repositorios seleccionados.</p> <p>Además, posiblemente tendrás que descargar también las claves de verificación de cada repositorio, pero el generador te lo pone muy fácil y te muestra una línea de comandos para copiar y ejecutar en un terminal para obtenerlas automáticamente.</p> <p>Si echas un vistazo a la web del generador, la verdad es que se han pegado una buena currada recopilando <strong>repositorios de aplicaciones populares</strong>, como puede ser AWN, Banshee, Chromium,  DropBox, los repos oficiales de Google y GetDeb, Medibuntu, Skype, VirtualBox…</p> <p>Vía: <a href="http://www.ubuntugeek.com/tipubuntu-sources-list-generator.html">UbuntuGeek</a>.</p> Arte conceptual para rediseñar pósters de series famosas http://www.hachemuda.com/2010/01/arte-conceptual-para-redisenar-posters-de-pelis-y-series-famosas Guillermo Latorre Guillermo Latorre película películas póster risa serie Mon, 18 Jan 2010 00:00:00 +0100 /2010/01/arte-conceptual-para-redisenar-posters-de-pelis-y-series-famosas <h1> </h1> <p>Leyendo los elementos compartidos de <a href="http://www.google.com/profiles/alexliam">Alexliam</a> descubro la web de <strong>Máximo Gavete</strong> y su entrada titulada [Rediseñando posters de películas][2].
