Quizá te interese concerlos:</p> <ul> <li>Cuando tienes varias ventanas abiertas en el escritorio, puedes enviar una de ellas al fondo con un solo click. Haz click con el botón central, el de la rueda, sobre la barra del título de la ventana y la mandarás al fondo. Si no tienes botón central, puedes hacerlo pulsando los dos botones a la vez, derecho e izquierdo.</li> <li>Existe un plugin llamado <strong>Nautilus Send To</strong> que crea un menú contextual al hacer click derecho sobre un archivo con la opción <strong>Enviar a</strong>, permitiendo enviar ese archivo o archivos seleccionados como adjunto de un correo electrónico a un contacto del Evolution, o a un contacto de Gaim si está conectado. Para instalarlo, desde un terminal ejecuta:<br /> <code>sudo apt-get install nautilus-sendto</code></li> <li>Para reiniciar el entorno gráfico (sin reiniciar el equipo) simplemente tienes que pulsar la combinación de teclas &#91;Ctrl&#93; &#91;Alt&#93; &#91;Backspace] y se reiniciará, yendo a la pantalla de inicio de sesión. Si quieres matar el servidor X completamente, desde una consola, por ejemplo [Ctrl&#93; &#91;Alt&#93; [F1], ejecuta el comando **sudo /etc/init.d/gdm stop **para detener el servidor gráfico, y **sudo /etc/init.d/gdm start **para iniciarlo de nuevo.</li> <li>Los submenús de Gnome (por ejemplo los del menú principal) <strong>están retardados por defecto</strong>, se toman un breve tiempo para aparecer. Por supuesto, puedes <strong>hacerlos aparecer instantáneamente</strong>, sin ningún retardo. Abre el editor de textos de Menú -> Accesorios-> Editor de textos y en un documento nuevo pega esta línea:<code>gtk-menu-popup-delay = 0</code>Guarda el fichero en tu directorio personal (/home/tu_nombre) con nombre <strong>.gtkrc-2.0</strong> (sí el nombre empieza por un punto, porque es un fichero de configuración y tiene que estar oculto). Reinicia Gnome como hemos comentado antes, pulsando &#91;Ctrl&#93; &#91;Alt&#93; [Backspace] e inicia sesión de nuevo. <strong>¡Alucina con la velocidad de los menús!</strong><br /> Si no los quieres tan rápido, edita el fichero y sustituye el 0 por un 50, o un 100… Prueba tú mismo y ponlo** como tú estés más cómodo**. Como el fichero está oculto, abre una ventana de Nautilus y pulsa &#91;Ctrl&#93; &#91;H&#93; para que te muestre todos los ficheros ocultos también, botón derecho y abrir con el editor de textos. Cambias el valor, guardas y reinicias Gnome de nuevo.</li> <li>La función &#91;Alt&#93; &#91;Tab&#93; permite <strong>cambiar entre las aplicaciones abiertas</strong> sin usar para nada el ratón, activando la ventana que se selecciona con su icono. Una función parecida a la anterior para <strong>cambiar entre ventanas abiertas</strong> es situar el cursor del ratón sobre uno de los elementos de la lista de ventanas y <strong>mover la rueda</strong> para arriba o para abajo para ir en un sentido o en el otro.</li> <li>** Cambiar los permisos** de un archivo es muy sencillo, haz click derecho sobre el archivo y ve a* propiedades*.
