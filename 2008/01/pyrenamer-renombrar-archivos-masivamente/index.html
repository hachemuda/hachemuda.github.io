    Pyrenamer: renombrar archivos masivamente

    Publicado por Guillermo Latorre el 18 de Enero de 2008

    Pyrenamer es un programa para el renombramiento masivo de ficheros. Con él, puedes renombrar grupos de archivos siguiendo expresiones regulares y usando las herramientas que te propone el programa. Te deja la posibilidad de ser tú quien diga los patrones y expresiones para realizar los cambios, pero también incluye algunas funcionalidades y ayudas que lo hacen mucho más sencillo. Incluso con ejemplos para que veas qué tipo de patrones puedes usar.

    He grabado un video usando Pyrenamer para renombrar algunas fotos de diferentes formas, para ver la flexibilidad de las expresiones regulares y de las herramientas que trae el programa:

    En la página oficial nos explican cómo renombrar fácilmente 5 capítulos de Jericho que se llaman:

    Jericho.S01E01.Pilot.HDTV.XviD-XOR.avi
    Jericho.S01E02.Fallout.HDTV.XviD-XOR.avi
    Jericho.S01E03.Four.Horsemen.HDTV.XviD-XOR.avi
    Jericho.S01E04.The.Walls.of.Jericho.HDTV.XviD-LOL.avi
    Jericho.S01E05.Federal.Response.HDTV.XviD-LOL.avi

    para que pasen a llamarse de una forma mucho más bonita:

    01×01 – Jericho – Pilot.avi
    01×02 – Jericho – Fallout.avi
    01×03 – Jericho – Four Horsemen.avi
    01×04 – Jericho – The Walls of Jericho.avi
    01×05 – Jericho – Federal Response.avi

    Instalación

    En Ubuntu tan sólo tienes que tirar de repositorios:

    sudo apt-get install pyrenamer

    Para otras distribuciones dirígete a la sección de descargas de la página oficial y descarga el paquete .tar.gz con el código fuente. Las** instrucciones para instarlo** son las de siempre: descomprime el fichero, accede al directorio que se ha creado desde un terminal y ejecuta:

    ./configure
    make
    sudo make install
