    CSS1K: demostración de qué puede hacerse con un 1K de CSS

    Publicado por Guillermo Latorre el 20 de Enero de 2012

    Hace unos años se puso muy de moda en el mundo de los desarrolladores web una página llamada CSS Zen Garden, un proyecto en el que diseñadores y maquetadores de cualquiera parte del mundo podían mostrar sus diseños para una página web con unos contenidos comunes. De esta forma, se podía visitar la misma web, con los mismos contenidos, pero con hojas de estilo diferentes. El objetivo era servir de inspiración a otros diseñadores, así como demostrar la potencia de CSS y las ventajas de separar el contenido del diseño gráfico.

    La evolución de las tecnologías web ha sido enorme en los últimos años, también la de las hojas de estilo, y la velocidad de carga de la página se ha convertido en un factor importantísimo para los sitios web. Partiendo de las mismas premisas que CSS Zen Garden, Jacob Rask decidió hace unos meses publicar el proyecto CSS1K: un sitio web con contenido fijo y diferentes diseños con hojas de estilo que no superan el Kilobyte de peso.

    Se trata de una demostración de fuerza, un variado ejemplo de qué puede conseguirse con muy poco código CSS.

    Las normas de participación son sencillas:

    • Puedes enviar solamente CSS para la plantilla HTML común.
    • Puedes enviar CSS comprimido de como mucho 1K (1024 bytes) en total.
    • Los prefijos de los diferentes navegadores no cuentan dentro del peso final. Si necesitas poner prefijos, envía el código sin ellos y el sistema los pondrá por ti.
    • No está permitido usar recursos externos, por ejemplo mediante @font-face, data o @import.
    • El código enviado será licenciado bajo la MIT License.

    Si quieres participar, sólo tienes que hacer un fork del proyecto en GitHub y enviar tu propuesta como un pull request especificando tu nombre, nombre del nuevo diseño y tu página web. Si no tienes cuenta de GitHub, también puedes enviar tu propuesta por e-mail.
