    Pentotype: software de prototipado como en un papel

    Publicado por Guillermo Latorre el 27 de Enero de 2012

    A través del Twitter de Mark Ellul he descubierto un nuevo proyecto que quiere **innovar en la forma en que prototipamos **nuestras aplicaciones.

    Al conceptualizar, al darle una forma a una idea de aplicación, es muy habitual utilizar el clásico papel y lápiz. Empezar a dibujar pantallas, detalles, flujos de información… También existe mucho software creado especialmente para el prototipado de aplicaciones y sitios web, con colecciones de elementos y plantillas, además de una interfaz y funcionalidades muy útiles para este tipo de trabajo.

    Pentotype quiere plantear una nueva forma de hacer las cosas: software de prototipado como en un papel. Quiere quedarse con lo mejor de utilizar el papel/lápiz y lo mejor de usar software de prototipado. Combina ambas metodologías en una nueva propuesta de la que todavía no podemos ver más que la idea principal.

    El video explica muy bien cuál es la problemática a resolver y el objetivo del proyecto:

    Dibujar como si fuera un papel, pero con una conversión directa a software. Dibujar como con un lápiz, pero que el sistema reconozca los elementos y los transforme y entienda como funcionales. Con la agilidad y flexibilidad que permite utilizar un software de prototipado.

    Pentotype todavía no es una realidad, es una idea en desarrollo con una lista de correo activa para que cualquier usuario pueda apuntarse y recibir información de su evolución. Si eres un profesional de este mundillo, al registrarte puedes marcar la opción para que puedan hacerte una entrevista rápida sobre tu metodología de trabajo, para ayudar en el desarrollo y mejorar Pentotype.
