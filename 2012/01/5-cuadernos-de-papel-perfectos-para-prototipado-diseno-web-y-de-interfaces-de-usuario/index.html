    5 cuadernos de papel perfectos para prototipado, diseño web y de interfaces de usuario

    Publicado por Guillermo Latorre el 30 de Enero de 2012

    Cada maestrillo tiene su librillo. En las profesiones del diseño web y aplicaciones, prototipado de interfaces y artes relacionados con la experiencia de usuario, cada uno tiene sus herramientas. Existe muchísimo software de escritorio y aplicaciones web dedicadas a estas tareas, pero en muchas ocasiones lo más efectivo es lo clásico: goma, papel y lápiz.

    Lógicamente depende de tu forma de trabajar y del tipo de proyecto que estés llevando a cabo, pero yo personalmente opino – y creo que no soy el único - que no hay nada más efectivo que **emborronar unas cuantas hojas **de papel. Y después ya veremos cómo se pasa eso a limpio,

    Voy a comentar cinco opciones para los aventureros que todavía llevan un lapicero blando y una goma de borrar en la mochila.

    App Sketchbooks: iPhone, iPad y Web

    La empresa estadounidense Square Position tiene editados unos cuadernos recomendados para diseñadores y desarrolladores de aplicaciones iOS y web. Se trata de libretas con anillas y tapa dura, con las hojas perforadas para poder quitarlas fácilmente.

    El dibujo de las hojas varía dependiendo del sabor de cuadernillo que escojamos:

    • iPad App Sketchbook: el tamaño y las siluetas de cada página son las del iPad, punteadas en la parte central como guía para el dibujo de los elementos de la pantalla.
    • iPhone App Sketbook: lo mismo, pero la versión iPhone. Hojas con tres siluetas de tres pantallas y espacio para anotaciones.
    • UX Sketchbook: un cuaderno de diseño general de sitios y aplicaciones web. Cada hoja tiene siluetas de un navegador web, con un grid para ayudar al dibujo y espacio para comentarios.
    • Responsive Design Sketchbook: sin duda el que más me gusta de esta empresa. Las siluetas están preparadas para pensar en las diferentes resoluciones y diseñar una interfaz adaptable a todas ellas.

    El precio es el mismo para todos ellos: 12,95$.

    Web Design Sketchbook

    El Web Design Sketchbook es un libro de bocetos pensado para diseñadores web. Sus páginas tienen impresas rejillas de ayuda para dibujar, además de prototipos de ventanas de navegadores web en diferentes tamaños y espacios libres para tomar notas.

    No lo he visto físicamente, pero viendo las fotos de la web este cuaderno parece más un libro o un álbum de fotos: tapas duras, gruesas y brillantes, acolchadas.

    El equipo que ha editado este libro también ha publicado un PDF con varias plantillas para descargar libremente desde la web.

    Su precio es el más caro de todas las opciones que comentamos en este artículo: 30$.

    Enlace:

    • Web Design Sketchbook

    UI Stencils Sketch Pads

    UI Stencils es una tienda online para diseñadores web y de aplicaciones para móviles. Se hicieron populares cuando mostraron al mundo su nueva idea: una regla de dibujo como las de toda la vida, pero con patrones de diseño de inferfaces de usuario en vez de formas geométricas. Es una regla que te ayuda a tomar medidas y a dibujar botones, iconos, formularios, etc.

    En la tienda puedes encontrar unos librillos de hojas con cuadrículas de ayuda para el dibujo y con algunas siluetas ya impresas: navegadores web, contornos de iPhone/iPad… e incluso puedes comprarlos como hojas normales o con adhesivo, para poder quitar las hojas y pegarlas donde quieras.

    Es un rollete artesano y simple, aunque para no tener tapas ni anillas creo que es un poquitín caro, comparándolo con otras soluciones. Para ver en detalle una hoja de los librillos, está disponible una plantilla de navegador web en A4 para descargar en PDF.

    Un librillo cuesta 13.95$ y te hacen descuento si compras varios a la vez.

    Las reglas cuestan 26.95$, también te hacen descuento por varias compras y viene con un lapicero.

    ActionBooks, la versión en papel del ActionMethod

    Otra empresa americana desarrolló hace tiempo una metodología de trabajo que llamó ActionMethod. Es un método para mejorar la productividad y el enfoque en las tareas importantes, capturándolas de la forma oportuna y priorizándolas sobre las demás. Las herramientas para los seguidores del método son una aplicación online, una versión de la app para móviles y para iPad… y una serie de productos en papel que son un auténtico alucine.

    En la tienda online puedes ver el catálogo completo. Tienen libretas pequeñas pensadas para la movilidad, paquetes de hojas y tarjetas basadas en su método de gestión de tareas, cuadernos tipo* Moleskine* con diferentes tapas y anillas…

    Pero la joya son el Dot Grid Book y el Action Book. El primero es un cuaderno un poco más grande que un A4 con un papel de gran grosor y textura, perfecto para dibujar, pintar y tomar notas. Con anillas resistentes y unas tapas duras muy elegantes que da gustito tocarlas.

    El segundo, el Action Book, es exactamente el mismo cuaderno que el Dot Grid, pero pensado para los seguidores de la metodología. En las caras frontales de las páginas, incluye un lateral con espacios para capturar las acciones a realizar y después recortarlas o trasladarlas donde quieras.

    Estos dos valen 15$ cada uno.

    Yo ya he hecho tres encargos de ActionBooks en los últimos 2 años. Fan total,

    Háztelo tú mismo

    El quinto y definitivo no es un cuaderno editado y preparado para que lo compres, es una sugerencia para hacértelo tú mismo.

    A mí personalmente me gusta la idea de comprar un cuaderno ya hecho, pensado y terminado para usarlo como entorno de pruebas, brainstorming y experimentos. Pero también existen en internet muchísimos artistas que han publicado plantillas en PDF listas para que cualquier usuario las pueda descargar e imprimir.

    Algunas de ellas:

    • Konigi: Graph paper
    • Jeremy P. Alford: responsive web design sketch sheets
    • Henk Wijnholds: A4 sketching paper template
    • PaperBrowser: sketchs downloads
    • iPhone Wireframe Templates
    • iPad Idea Sheet y iPad Design Template Stencils

    Por tanto, otra opción igual (o más) interesante es buscar las plantillas que más se adapten a tu trabajo e imprimirlas tú mismo. Y ya si se quiere hacer para nota, se pueden preguntar en una imprenta los precios y opciones para imprimir libros de notas, cuadernillos de hojas, encuadernaciones, con tapas, etc.

    Elige un papel bueno, de buen grosor y textura. Imprime algunas buenas. Prueba.
