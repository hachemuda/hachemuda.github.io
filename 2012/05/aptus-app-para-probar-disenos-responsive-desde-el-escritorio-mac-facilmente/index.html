    Aptus: app para probar diseños responsive desde el escritorio Mac fácilmente

    Un navegador web para Mac que permite configurar diferentes resoluciones de pantalla para realizar pruebas rápidas de diseños web adaptables.

    Publicado por Guillermo Latorre el 15 de Mayo de 2012

    En este blog ya hemos publicado varios artículos sobre herramientas para el desarrollo de sitios web responsive, como la recopilación de 5 servicios web para probar el diseño adaptable o de 5 cuadernos de papel para diseñadores.

    En esta ocasión, me gustaría hablar de Aptus, un navegador para entornos Mac que permite visualizar sitios web en diferentes resoluciones de una manera muy cómoda y sencilla. Es una aplicación de escritorio en la que el usuario puede configurar los tamaños de pantalla que quiera, creando botones en la parte superior para poder visualizar la web en cualquiera de ellos.

    Yo la llevo usando unas cuantas semanas y estoy muy contento, porque permite una forma muy rápida y ágil de cambiar la visualización entre varias resoluciones.

    Para cada nuevo tamaño, Aptus permite elegir si debe tener un comportamiento fijo o fluído, especificar la resolución inicial en pixels y elegir un icono para encontrarlo rápidamente en el carrusel superior.

    Aptus: probando el inspector de elementos.

    Además, está integrado con el inspector de elementos de WebKit, por lo que puede consultarse el código y debuggear directamente desde el navegador, como si fuera Safari.

    Como bola extra, Aptus viene de serie con un capturador de imágenes, para poder hacer capturas de todo el sitio web con un solo click.
