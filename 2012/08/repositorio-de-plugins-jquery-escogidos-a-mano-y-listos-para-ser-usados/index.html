    Repositorio de plugins jQuery escogidos a mano y listos para ser usados

    Publicado por Guillermo Latorre el 30 de Agosto de 2012

    Handpicked jQuery Plugins es un repositorio web con el soporte de Cloudflare y MaxCDN con una colección de plugins de jQuery escogidos manualmente por su utilidad y calidad.

    Desde la propia web puedes ver todos los plugins incluídos en el repositorio y descargar cada uno de ellos. Incluso tienes disponibles las URL’s de cada plugin en un CDN, para que puedas usarlos libremente sin ni siquiera tener que descargarlos.

    Todos los plugins están preparados para ser ejecutados directamente, puedes descargar el repo a tu ordenador y hacerlos funcionar localmente sin modificar nada, con todos los ficheros Javascript y CSS comprimidos con YUI compressor, optimizados para que la carga sea la menor posible en tu web.

    En cuanto a los propios plugins, los hay de todo tipo. Una gran colección de complementos para crear efectos visuales, funcionalidades especiales para formularios, menús, creación de carruseles y sistemas de ventanas emergentes, integración con redes sociales… Vamos, un montón de recursos,

    El sitio web es feo de cojones pero enormemente funcional y se agradece mucho la selección personal de los plugins. También dispones de un repositorio de GitHub con todos los ficheros actualizados.
