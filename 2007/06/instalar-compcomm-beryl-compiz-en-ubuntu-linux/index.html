    Instalar CompComm (Beryl + Compiz) en Ubuntu Linux

    Publicado por Guillermo Latorre el 20 de Junio de 2007

    El artículo original en el que está basado este post está en el foro de OpenCompositing, pero lleva un tiempo caído y no se puede acceder a él. En el post de ayer pudiste ver un video de CompComm y sus efectos en acción, así que ahora toca instalarlo y probarlo tú mismo.

    Esto es un tutorial de instalación de CompComm (Composite Community), que es el proyecto formado por la unión de Beryl y Compiz. El proceso es muy sencillo gracias a un script de instalación que salió del mismo foro y ahora cuelgo yo aquí para que se pueda descargar sin problemas. El proceso de instalación funciona tanto para Gnome como para KDE, aunque yo sólo lo he probado en Gnome.

    Vamos al tema:

    1. Descarga el script makeccc (click derecho -> guardar enlace como…) *y guárdalo en tu disco duro con el nombre *makeccc.
    2. No sé si es necesario o no, pero yo mejor desinstalaría Beryl antes de nada:
      sudo apt-get remove --purge beryl beryl-plugins beryl-manager
    3. Abre un terminal y accede al directorio donde has descargado el script. Dale permisos de ejecución:
      chmod x makeccc
    4. Instalar los paquetes necesarios:
      ./makeccc packages
    5. Descargar/actualizar Compiz/CompComm:
      `./makeccc clone“`
    6. Instalarlo:
      ./makeccc install
    7. Si quisieras desinstalarlo puedes hacerlo ejecutando:
      ./makeccc uninstall
    8. Configurar CompComm. Tienes dos opciones (yo uso la primera). Una, ejecutando el comando:
      ccsm (o desde Sistema -> Preferencias -> CompizConfig Settings Manager)
      La segunda, ejecutando el comando:
      ccs-settings (o desde Sistema -> Preferencias -> CCS Settings Manager)

    En el panel del CompizConfig Settings Manager puedes configurar absolutamente todos los parámetros que quieras. Están bastante bien ordenados y accesibles, seguro que te pasas un ratito cambiando y curioseando cosas,

    Ya puedes ver tu nuevo escritorio con CompComm funcionando. Abre un terminal y ejecuta:

    compiz-manager

    ¡Y a disfrutar! Si quieres que se active CompComm siempre que inicies sesión, no tienes más que añadir el comando compiz-manager como programa al inicio (en Gnome desde Sistema -> Preferencias -> Sesiones).

    Mucho cuidado: no actives el plugin de GLib (en la sección Utility) porque debe estar dando muchos problemas con el gestor de preferencias.
