    KeyJNote: presentaciones en PDF con efectos impactantes

    Publicado por Guillermo Latorre el 07 de Junio de 2007

    KeyJNote es un script para conseguir hacer presentaciones brillantes y llenas de efectos espectaculares sobre un simple documento en PDF. Está disponible para GNU/Linux, Windows y Mac.

    Genera el documento de la presentación en formato PDF con el programa que desees, y después usa el script para realizar una presentación elegante y con todo lujo de detalles.

    El video con la demostración lo ha grabado Cristian:

    Entre los efectos que es capaz de hacer, están:

    • Muchos efectos diferentes para las transiciones entre páginas.
    • Reorganización de las diferentes páginas para ir directamente a una.
    • Zoom circular acompañando al ratón, oscureciendo el resto de la pantalla.
    • Iluminar zonas seleccionadas del documento, y oscurecer el resto

    Instalación:

    Deberás tener instalados los paquetes de Python para que funcione. En Ubuntu, ejecuta la línea:

    sudo apt-get install python python-opengl python-pygame python-imaging gs pdftk

    Descarga el programa. Descomprímelo, accede al directorio descomprimido desde un terminal y copia el script al directorio /usr/bin ejecutando el comando:

    cp keyjnote.py /usr/bin/

    Ya está instalado. Para visualizar un documento con el KeyJNote deberás ejecutar el comando:

    keyjnote.py nombre_del_archivo.pdf

    Si no se visualiza correctamente, prueba a desactivar Beryl. Parece que no se llevan muy bien,
