    Videotutorial Gimp: crear llamas y texto de fuego

    Publicado por Guillermo Latorre el 02 de Mayo de 2007

    En este videotutorial explico cómo crear llamas de fuego con el Gimp, y después un ejemplo de cómo usarlas para crear un texto con una textura de fuego.

    []: http://hachemuda.com/uploads/videotutoriales/VideoTutorialGimp-textoFuego.avi “Descarga directa”

    Creado íntegramente con software libre y sobre Ubuntu Feisty, lo he grabado usando RecordMyDesktop y editado con el propio Gimp y Kino.

    La gracia del proceso está en crear las llamas, y después poder usarlas para cualquier cosa. Parece un poco complicado pero es muy sencillo, se trata simplemente de lanzar un degradado de los colores del fuego: amarillo, naranja y rojo. Después, podemos usar* el filtro de deformaciones* para dar forma a las llamas en unos poquitos clicks, y teniendo todo el control sobre el resultado.

    Puedes descargártelo a tu disco duro (en formato AVI y a mayor calidad) pinchando en la imagen de arriba, o** también puedes verlo online**:
