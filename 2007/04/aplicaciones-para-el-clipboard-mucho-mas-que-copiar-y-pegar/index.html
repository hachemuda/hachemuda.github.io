    Aplicaciones para el clipboard: mucho más que copiar y pegar

    Publicado por Guillermo Latorre el 16 de Abril de 2007

    Manejar bien el arte del copy & paste es fundamental a estas alturas. Si usas Gnome y res un artista del clipboard y se te quedan cortas las combinaciones de teclas [Ctrl] [C] y [Ctrl] [V], estas sencillas aplicaciones pueden hacerte la vida mucho más fácil.Olvídate de los líos de qué tienes copiado en este momento, si está guardando el texto que has seleccionado antes con el ratón o lo que has copiado después, qué ocurrirá si pulso el botón del medio del ratón…

    Estas aplicaciones intentan facilitarte la experiencia con el portapapeles y hacer del Copiar y Pegar un acto placentero y seguro.

    Parcelle

    Este es el que más me gusta. Su instalación es sencilla y está completamente integrado en el escritorio Gnome, usando los mensajes del programa del mismo modo que los mensajes de sistema.

    Mantiene una ventana completa con todas las entradas que se tienen copiadas para poder trabajar con ellas. Quizá le falta algo para hacerlo más rápido en el uso cotidiano, personalizar una combinación de teclas, por ejemplo. Es muy bueno, te recomiendo que lo pruebes.

    Para instalarlo, abre un terminal, y entra al directorio en el que quieras tener la carpeta parcelle de instalación del programa. Ejecuta el comando:

    svn co http://xyhthyx.googlecode.com/svn/trunk/parcelle parcelle

    Esto te crea un directorio llamado parcelle. Accede a él tecleando cd parcelle e instala el programa ejecutando el comando:

    sudo python setup.py install

    Ya está. Ya puedes ejecutarlo yendo a Menú -> Aplicaciones-> Accesorios -> Parcelle.

    Desktop Data Manager

    Es un poco más completo que los otros dos, porque incluye un capturador de pantalla y un gestor de descargas, además del propio gestor del portapapeles. Permite configurar las combinaciones de teclas que harán aparecer una especie de menú emergente donde elegir las entradas de cosas copiadas para poder trabajar con ellas. La gestión de las imágenes como simples ficheros es una pasada.

    Si estás en Ubuntu Edgy **o **Feisty seguramente necesitarás instalar primero un paquete necesario para que funcione el** Desktop Data Manager**. Hazlo mediante el comando:

    sudo apt-get install libmono-system-runtime1.0-cil

    Para Ubuntu, descarga el paquete .deb para tu arquitectura e instálalo haciendo doble click sobre él o usando el comando sudo dpkg -i nombre_paquete.deb *si quieres hacerlo en terminal. Te creará una entrada llamada Desktop Data Manager en el *Menú -> Aplicaciones -> Accesorios.

    En la página oficial del DDM se han currado unos screencasts en los que puedes ver cómo funciona el programa.

    Glipper

    Es el más rápido y sencillo de los tres, guarda las entradas de las cosas que se van copiando y seleccionar una de ellas es cuestión de dos clicks de ratón o usando la combinación de teclas para que salga el menú emergente. Si juegas mucho al copy & paste, Glipper puede ser un buen aliado.

     

    En Ubuntu, puedes instalarlo desde los repositorios ejecutando el comando:

    sudo apt-get install glipper

      • *Si te gusta cualquiera de ellos, puedes hacer

    que se ejecute cada vez**** que inicies una sesión, y así no tener que preocuparte de activarlo cuando lo necesites. Para que se ejecute automáticamente, ve a Sistema -> Preferencias -> Sesiones y en la pestaña de programas al inicio añade uno nuevo, poniendo como comando uno de los tres que estamos comentando: parcelle, glipper o desktop-data-manager.
