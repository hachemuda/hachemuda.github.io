    Galerías y pases de diapositivas con javascript para la web

    Publicado por Guillermo Latorre el 19 de Febrero de 2007

    He tenido que hacer un slideshow en una página web, y buscando por la red me he encontrado con un montón de ejemplos y de scripts que hacían exactamente lo que yo buscaba, disponibles para descargar y usar (y de alguna manera agradecer al autor su buen trabajo).

    Crear tu propio álbum de fotos online y que quede profesional y elegante no es nada complicado gracias a todos estos programadores, que han creado plantillas para que sea lo más sencillo posible crear una bonita galería.

    Nada de flash, nada de programas extraños… Sólo un editor de texto, HTML, CSS y Javascript. Estos son algunos scripts que a mí me gustan mucho:

    • Slideshow: una librería javascript sobre MooTools. Esta librería es una pasada, permite aplicar efectos geniales para las transiciones entre las fotografías. Puedes personalizar tu pase de diapositivas con los efectos que tú quieras, incluso combinándolos. Su página web te muestra todos los efectos y posibilidades que tiene. Lo mejor es su grado de adaptabilidad y personalización.

    • Couloir.Org: slideshow con redimensionamiento y efecto fade en las transiciones. Este script viene genial para imágenes de diferentes tamaños que no quieras adaptar a la presentación, porque con un efecto muy agradable y elegante el marco de la imagen se redimensiona para mostrar correctamente cada una de las tomas. Además queda muy limpio con el efecto de atenuación al pasar de una a otra.

    • Image Cross Fader Redux. Es un script que muchos otros han usado para crear el efecto de atenuación en la transición entre las imágenes. El resultado final es genial, muy limpio y sencillo. Funciona con un temporizador para cada imagen.

    • JonDesign’s: SmoothGallery. Usando este script puedes crear una galería sensacional en tan sólo unos minutos. Permite establecer las transciones mediante un temporizador o con clicks de ratón. Este es el que más me gusta, aunque las imágenes de las flechitas para moverse entre las fotos no le quedan muy bien. Muy elegante. [![Pantallazo de la página][8]][8]
