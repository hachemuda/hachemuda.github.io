    PoEdit: ayuda para la traducción de programas

    Publicado por Guillermo Latorre el 21 de Julio de 2007

    PoEdit es un programa que te ayuda en el proceso de traducción de cualquier programa. Si alguien escribe un programa, puede dejar el fichero del catálogo de traducción (con extensión .po) en la web y cualquier persona se lo descarga, lo traduce a su idioma y se lo envía al autor. Me parece bastante necesario tener un gestor de catálogos de traducción de este tipo, porque te facilita muchísimo las cosas.

    Yo lo he usado hace poco para realizar la traducción del programa [GRemind][3], el gestor de recordatorios del que ya hablé en HacheMuda. Funciona muy bien, es realmente cómodo y sencillo de usar.

    [3]: http://perticone.homelinux.net/~sergio/c /gremind/ “Página oficial del proyecto GRemind” Para instalarlo en Ubuntu Linux basta con tirar de repositorios:

    sudo apt-get install poedit
